# mobile network server

`$ pip install -e .` - установить пакет  
`$ python tests/tests.py` - прогон тестов

пример запуска программы:
```
$ run_counter -i ./tests/request.json
>> 70
```

формат входных данных:  
`count_unreachable` принимает на вход путь до json-фаила, представляющему массива из 4 объектов 
(по объекту для каждого входного
параметра):
1. Объект, описывающий структуру сети в виде массива массивов (каждый из которых
состоит из пары строк). В строке указано имя вершины.
2. Объект, описывающий пользователей на каждой вышке, представляется в виде dict, где
ключи имя вышки, а значения - количество пользователей.
3. Объект типа строка, описывающий название вышки, отправляющей экстренные
сообщения
4. Объект, описывающий массив из строк, где каждая строка — это имя разрушенной вышки.

докер:
`docker build --tag=mariamsu/asvk_server:latest --tag=mariamsu/asvk_server:2 .` - сбилдить
`docker run --rm -p 5019:5019 -e HOST="0.0.0.0" -e PORT=5019 -it mariamsu/asvk_server` - запустить
